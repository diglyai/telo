# Voke SDK (Node.js)

The Node.js SDK provides the authoring surface for Voke modules. It defines the shared contracts (types and lifecycle interfaces) that modules use to plug into the runtime, so module code stays consistent across implementations.

## What It Provides

- **Module interfaces** for registering resources and handling execution.
- **Context types** for accessing the kernel, registry, and events.
- **Shared primitives** used by Voke modules and tooling.

## Status

Early prototype. APIs and contracts are still evolving. The API surface - including YAML shapes - may change at any time without notice.

## When to Use It

Use the SDK when building or extending Voke modules. It is not the runtime itself; it is the contract layer that keeps module behavior consistent and predictable.
